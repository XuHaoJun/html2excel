# html2excel

| HTML              | Excel             |
| ----------------- | ----------------- |
| `<strong>`, `<b>` | bold              |
| `<em>`, `<i>`     | italic            |
| `<ins>`, `<u>`    | underline(single) |
| `<br>`            | newline           |
